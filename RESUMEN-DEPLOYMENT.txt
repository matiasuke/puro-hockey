════════════════════════════════════════════════════════════════════
   TROUBLESHOOTING
════════════════════════════════════════════════════════════════════

¿Qué si falla el build?
→ Verifica: npm run build (local)
→ Si funciona local, el deployment funcionará

¿Qué si no obtiene URL de Vercel?
→ Ve a: https://vercel.com/dashboard
→ Busca proyecto: puro-hockey
→ Copia manualmente la URL

¿Qué si Railway no conecta?
→ Verifica DATABASE_URL = ${{Postgres.DATABASE_URL}}
→ Verifica CORS_ORIGIN = tu URL de Vercel
→ Espera 5-10 minutos al deploy

¿Qué si falla la conexión?
→ Verifica VITE_API_URL en Vercel
→ Verifica que Railway está corriendo (verde)
→ Redeploy en Vercel
